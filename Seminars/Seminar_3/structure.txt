//3.Квадрат
//4.Круг
//У каждой из этих фигур реализовать методы вычисления периметра и площади этих фигур.
// Так же для каждой фигуры реализовать проверку на корректность данных при создании фигуры,
// то есть можем ли мы начертить такой треугольник с заданными стонами или квадрат или
// прямоугольник. Так же круг может существовать только с радиусом больше нуля.
// Использовать эти классы в main и проверить все методы и конструкторы классов.

1. Создаем абстрактный класс Figure c пустым конструктором + 2 абстрактных метода (площадь, проверка)
2. Создаем интерфейс периметр
3. Создаем класс треугольник, наследуется от фигуры и интерфейс периметр:
    поля: int a, int b, int c - стороны треугольника
    Конструктор с тремя полями
    Проверка1:  a > 0 && b > 0 && c > 0
    Проверка2: a + b > c && a + c > b && c + b > a
    Периметр: p = a + b + c
    Площадь: s = sqrt(p/2 * (p/2-a)*(p/2-b)*(p/2-c))
4. Создаем класс прямоугольник, наследуется от фигуры и интерфейс периметр:
    поля int a, int b
    Конструктор 1: 2 поля a, b
    Проверка 1: a > 0 && b > 0
    Проверка 2: a < a + 2 * b && b < b + 2 * a
    Периметр 2 * (a+b)
    Площадь а * b
5. Создаем класс квадрат, наследуется от прямоугольник и интерфейс периметр:
    Конструктор int a : int b = a
6. Создаем класс круг, наследуется от фигуры
    Поля int r
    Проверка r > 0
    Длина окр 2*pi*r
    Площадь pi * r**2
